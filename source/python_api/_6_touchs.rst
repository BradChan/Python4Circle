====================
触摸输入的API
====================

1) 检查A1触摸盘的状态：touch_A1；有效值为True或False，当A1触摸盘被触摸时为True，否则为False。用法如下：

.. code-block::  python
  :linenos:

  import time
  from  hiibot_circle  import  hbc

  while True:
      if hbc.touch_A1:     # 判断A1触摸盘是否被触摸
          print("A1 be touched")
      time.sleep(0.1)
